## 🛠️ Tecnologie Utilizzate

- **Frontend**: HTML5, TailwindCSS (CDN)
- **JavaScript**: Vanilla JS
  - Mobile hamburger menu toggle
  - Scroll reveal animations (Intersection Observer)
  - Interactive image slider with keyboard navigation
  - Gallery configuration loading from JSON
- **Styling**:
  - Dark theme (#1A1A1A background)
  - Gold accents (#C4A77C)
  - Slate secondary text (#4F4F4F)
  - Responsive rounded borders (rounded-2xl standardized)
- **Fonts**:
  - Playfair Display (Serif) - headings
  - Inter (Sans-serif) - body text

## 📱 Responsive Design

- **Mobile First**: Ottimizzato per dispositivi piccoli
- **Breakpoints**:
  - sm: 640px
  - md: 768px
  - lg: 1024px
  - xl: 1280px
- **Features**:
  - Hamburger menu mobile
  - Adaptive typography
  - Flexible grids (1 col → 2 col → 4 col)
  - Fixed footer su desktop
  - Sticky navbar su tutte le pagine
